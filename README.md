# Модуль фитнес-трекера

## Получая информацию от датчиков, модуль позволяет получать следующие данные о ваших тренировках:
- Тип тренировки;
- Продолжительность тренировки;
- Дистанция;
- Скорость;
- Потраченные калории.

## Поддерживаются следующие виды тренировок:
- Бег;
- Спортивная ходьба;
- Плавание.
